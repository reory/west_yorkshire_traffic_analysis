# Contributing to West Yorkshire Traffic Analysis

First off, thank you for considering to contribute to this project and my others! 

### How can I contribute?
* **Reporting Bugs:** Open an issue and describe the bug.
* **Feature Requests:** Have an idea for a new chart or filter? Open an issue!
* **Pull Requests:** 1. Fork the repo.
  2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
  3. Commit your changes.
  4. Push to the branch.
  5. Open a Pull Request.

### Development Setup
1. Clone the repo.
2. Install requirements: `pip install -r requirements.txt`.
3. Run the dashboard: `streamlit run app.py`.
